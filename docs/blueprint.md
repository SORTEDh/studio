# **App Name**: Patient Support MVP

## Core Features:

- Secure User Authentication: Implement user authentication with JWTs for secure access.
- Patient Information Storage: Store and manage patient data using Prisma and SQLite.
- Prescription Image Upload: Allow patients to upload prescription images to Cloudinary or S3-compatible bucket.
- Automated Email Notifications: Send automated email notifications via Resend/SendGrid for prescription updates and reminders.
- AI-Powered Prescription Analysis Tool: Analyze uploaded prescription images using generative AI to extract relevant information, acting as a tool for staff.
- Multilingual Support: Enable multilingual support using react-i18next.
- Admin Dashboard: Provide an admin dashboard to manage users and prescriptions.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to evoke trust and serenity.
- Background color: Light gray (#F0F4F8), a desaturated version of the primary, for a clean and unobtrusive backdrop.
- Accent color: Soft green (#81C784), an analogous hue, for positive reinforcement and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for both headlines and body text.
- Use clear and professional icons from a library like FontAwesome.
- Maintain a clean, responsive layout using Tailwind CSS.
- Use subtle transitions and animations to improve user experience.
